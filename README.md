# IEDC VISAT Official Website üöÄ

Welcome to the official repository for the Innovation and Entrepreneurship Development Cell (IEDC) of VISAT Engineering College. This website serves as the digital face of our community, showcasing our events, team, and gallery while providing a way for students and partners to connect with us.

## üåü What We Built & Why

We created this platform to move beyond a static informational site. We wanted an **interactive, premium, and modern experience** that reflects the innovative spirit of IEDC.

### Key Integrations

1.  **Interactive User Interface (UI)**:
    *   **What it is**: Smooth page transitions, staggered menu animations, and a "Prismatic Burst" background effect.
    *   **Why**: To grab attention instantly and make browsing feel fluid and high-tech.

2.  **Dynamic Event Management**:
    *   **What it is**: A system to easily list "Upcoming" and "Past" events with posters and descriptions.
    *   **Why**: To keep the community updated without needing complex code changes every time.

3.  **Smart Contact Form (EmailJS)**:
    *   **What it is**: A contact form that sends emails directly to our inbox (`iedcvisat25@gmail.com`) without needing a backend server.
    *   **Why**: It simplifies communication. No server maintenance is required to receive messages, making it cost-effective and reliable.

4.  **Google Maps Integration**:
    *   **What it is**: An embedded map pointing directly to the IEDC Room at VISAT Campus.
    *   **Why**: To help visitors and students locate our physical innovation hub easily.

### üé® Signature UI Components

We didn't just use standard buttons and boxes. We integrated specific high-end components to give the site a distinct personality:

1.  **Masonry Layout (The Gallery)**:
    *   **What it is**: Instead of forcing all images into identical squares (which looks boring), the Masonry layout stacks them like bricks in a wall, respecting the original aspect ratio of each photo.
    *   **Why it Suits Us**: It mimics an art gallery wall. It allows us to showcase event photos in their full glory‚Äîwhether portrait or landscape‚Äîcreating a dynamic, organic visual experience that feels "curated" rather than "generated."

2.  **Prismatic Burst (The Background)**:
    *   **What it is**: A subtle, animated background effect that creates a sense of depth and energy without distracting from the text.
    *   **Why it Suits Us**: It represents "Innovation" and "Energy." The movement keeps the site feeling alive, separating us from static, outdated college websites.

3.  **Tilted Cards (The Events)**:
    *   **What it is**: Event cards that tilt and shimmer in 3D when you hover over them.
    *   **Why it Suits Us**: It adds *tactile* interactivity. It encourages users to "touch" and explore the events, turning a simple list into an engaging playground.

4.  **Staggered Menu**:
    *   **Why it Suits Us**: It creates a feeling of "polish" and smooth motion. It shows attention to detail‚Äîa core value of any good engineer or entrepreneur.

5.  **Decrypted Text (The Loading Screen)**:
    *   **What it is**: A matrix-style text reveal animation where random characters scramble and decode into the final text ("IEDC" and "VISAT") upon loading.
    *   **Why it Suits Us**: It evokes a sense of technology, cryptography, and code‚Äîfitting perfectly for an innovation and engineering cell. It sets a futuristic, high-tech tone immediately when the site loads.

## üöÄ How to Run This Project

If you want to run this website on your own computer:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Startups-IEDC-VISAT/IEDC-VISAT-Website.git
    cd IEDC
    ```

2.  **Install the "Parts" (Dependencies)**:
    ```bash
    npm install
    ```

3.  **Start the Engine**:
    ```bash
    npm run dev
    ```
    Open the link shown (usually `http://localhost:5173`) in your browser.
